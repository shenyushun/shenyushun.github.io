生成斐波那契数列并取前10项


def func(m):
  n,a,b = 0,1,1
  while n < m:
    yield a
    a,b = b,a+b
    n += 1

for one in func(10):
  print one

这个可以说是一道常见的简单算法题了，关键点就是理解a,b=b,a+b以及yield的作用。
