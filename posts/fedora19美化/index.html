https://extensions.gnome.org

安装 Gnome Tweak tool 调出最大化和最小化按钮以及更改主题

yum install gnome-shell-theme* 安装所有主题

如果不能使用shell主题 打开Tweak tool中shell扩展的user thems

安cairo-dock

安装鼠标主题:在终端中输入命令：sudo yum search cursor-theme，找喜欢的安装。

安装图标主题:与安装鼠标主题一样，在终端中输入命令：sudo yum search icon-theme找喜欢的安装。
