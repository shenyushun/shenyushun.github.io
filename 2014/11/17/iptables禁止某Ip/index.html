iptables禁止某Ip

11月 17, 2014 发布在 Centos

今天登录服务器一看，好家伙：

这尼玛绝对是非正常现象，想查看各个链接状态可以用下面的语句

结果如下：

我就呵呵了，看了一下这个ip，米国的，发现blocklist上面这个IP也上榜了

没啥说的，直接封IP吧，执行
iptables -I INPUT -s ***.***.***.*** -j DROP

如果想解封某ip,把I换成D即可。

最后，可以使用
iptables --list
查看当前规则列表。
