Roy
2016-06-15
зј–зЁӢжҠҖжңҜ

жӣҙж–°дәҺ2017.01.10

йҰ–е…Ҳе®үеҲ©жң¬д№ҰгҖҠtwo scoops of django1.8гҖӢзӣ®еүҚиҝҳжІЎжңүдёӯж–ҮзүҲпјҢдёҚиҝҮжҳҜжҲ‘зңӢиҝҮе…ідәҺdjangoзҡ„д№Ұдёӯж”¶иҺ·жңҖеӨ§зҡ„дёҖжң¬гҖӮжң¬жқҘжңүжңәдјҡе’ҢдҪңиҖ…йқўеҹәпјҢеҸҜжғңжҲ‘еӨ§еӨ©жңқзҡ„зӯҫиҜҒеӨӘйҡҫеҠһдәҶпјҢеҸӘеҫ—дҪңзҪўгҖӮ

иҝҷйҮҢжҺЁиҚҗдёүж–№еә“зҡ„еӨ§йғЁеҲҶеҮәиҮӘиҝҷжң¬д№ҰпјҢз§»йҷӨдәҶйғЁеҲҶеӨұж•ҲжҲ–иҖ…дёҚз»ҙжҠӨзҡ„еә“пјҢж·»еҠ дәҶдёҖдәӣе…¶д»–еә“пјҢејәзғҲе»әи®®еӨ§е®¶йҳ…иҜ»еҺҹд№ҰгҖӮ

дёӢйқўзҡ„жЁЎеқ—жңүдәӣжҳҜdjangoжЁЎеқ—пјҢжңүдәӣеҲҷжҳҜpythonжЁЎеқ—пјҢеҸҜд»Ҙи„ұзҰ»djangoзӣҙжҺҘдҪҝз”ЁгҖӮ

иҜқиҜҙйӮЈдәӣеӨҚеҲ¶еҲ°еҲ«ең°ж–№з„¶еҗҺж ҮжіЁвҖңеҺҹеҲӣвҖқзҡ„пјҢж•ўеҠ дёӘжң¬ж–Үй“ҫжҺҘдёҚпјҹжүӢе·Ҙж•ҙзҗҶеҫҲзҙҜзҡ„еҘҪдёҚпјҹ

sysctl -p

*nat
:PREROUTING ACCEPT [0:0]
:INPUT ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
:POSTROUTING ACCEPT [0:0]
-A PREROUTING -d иҪ¬еҸ‘жңҚеҠЎеҷЁзҡ„ip -p tcp --dport иҪ¬еҸ‘жңҚеҠЎеҷЁзҡ„з«ҜеҸЈ -j DNAT --to-destination зӣ®ж Үipпјҡз«ҜеҸЈ
-A POSTROUTING -d зӣ®ж Үip -p tcp --dport зӣ®ж Үз«ҜеҸЈ -j SNAT --to иҪ¬еҸ‘жңҚеҠЎеҷЁзҡ„Ip
COMMIT

дә§з”ҹиҝҷдёӘй—®йўҳзҡ„еҺҹеӣ еңЁдәҺжҲ‘зҡ„monky_patch()жҳҜеңЁзҲ¬иҷ«жЁЎеқ—дёӯжү§иЎҢпјҢиҖҢеёҢжңӣиҝҷдёӘзҲ¬иҷ«жЁЎеқ—иғҪеӨҹдҝқжҢҒзӢ¬з«ӢжҖ§дёҚеёҢжңӣе’ҢdjangoиһҚеҗҲзҡ„еӨӘж·ұпјҢжүҖд»Ҙи§ЈеҶій—®йўҳеҸӘйңҖж №жҚ®е®һйҷ…йңҖжұӮеңЁmanage.pyжҲ–settings.pyжҲ–wsgi.pyдёӯе…ҲжҠҠmonkey_patch()жү§иЎҢдәҶеҚіеҸҜгҖӮ
