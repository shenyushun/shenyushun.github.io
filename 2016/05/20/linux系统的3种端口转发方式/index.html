SSH

先说几个常用参数

  1. -q 静默运行，就算出错也啥都不提示
  2. -f 后台运行
  3. -C 压缩，带宽很差的时候压缩会快一些，否则会更慢
  4. -l 指定用户名，其实就是@前面写的那个
  5. -N 不执行任何命令，适合仅转发的情况
